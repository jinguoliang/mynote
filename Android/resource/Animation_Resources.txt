====== Animation Resources ======
Created 星期三 01 十月 2014

=== Property Animation ===

frameworks/base/core/java/android__/__animation
Animator
	|
	|---------------->AnimatorSet
	|---------------->ValueAnimator----------->ObjectAnimator

	1. res/animator/filename.xml
	2. ValueAnimator, ObjectAnimator, or [[AnimatorSet]]
	3. ObjectAnimator 对某个__对象__的属性进行修改
		a. AnimatorSet set = (AnimatorSet) AnimatorInflater.loadAnimator(myContext, R.anim.property_animator); 
		b. set.setTarget(myObject);
		c. set.start();
	4. ValueAnimator 对某个值改变如果想使用这个值，可以添加监听来
		a. ValueAnimator va=ValueAnimator.ofFloat(0,1);
		b. va.setDuration(1000);
		c. va.__addUpdateListener__(new ValueAnimator.__AnimatorUpdateListener__() {
		                        @Override
		                        public void onAnimationUpdate(ValueAnimator __valueAnimator__) {
		                            float value = (Float)valueAnimator.getAnimatedValue();
		                            Log.e(TAG, "the value is " + value);
		                        }
		                    });
		d. va.start();


=== View Animation  ===
frameworks/base/core/java/android__/view/__animation/
Animation
	|
	|---------------->AnimationSet

	1. res/anim/filename.xml
	2. AnimationSet, Animation       
	3. 加载使用
		a. ImageView image = (ImageView) findViewById(R.id.image);
		b. Animation hyperspaceJump = __AnimationUtils__.loadAnimation(this, R.anim.hyperspace_jump);
		c. image.startAnimation(hyperspaceJump);
	4. __Interpolator__  可以控制属性如何变化，是加速，减速，先加后减。。。
		a. [[AccelerateDecelerateInterpolator]]  --------------   @android:anim/accelerate_decelerate_interpolator 先加后减
		b. [[AccelerateInterpolator]]	-----------------------------   @android:anim/accelerate_interpolator	加速
	5. s.__setInterpolator__() or xml 指定
	6. __自定义Interplolator__
		a. <overshootInterpolator xmlns:android="http://schemas.android.com/apk/res/android"    overshootInterpolator是系统的
		    android:tension="7.0"			是我们可以修改它的张力
		    />
